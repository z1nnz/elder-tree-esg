# Elder Tree ESG Concept Brief

## 核心轉向

本專案不主打「讓使用者玩地圖探索遊戲」，而是以低壓力生活任務促進長者日常活動，並透過家庭樹、個人樹與 ESG 植樹回饋，讓使用者的日常行動累積成可見的陪伴成果與公益成果。

遊戲化元素保留，但定位為誘因與包裝，不是最終目的。

## 參考邏輯

類似 Unrot 這類習慣任務 App 的機制可以作為任務驗證參考：

- 系統派發低門檻任務。
- 使用者完成散步、閱讀、喝水、呼吸、專注、運動、拍照等任務。
- 系統透過照片、紀錄、感測資料或簡單回報判斷任務是否完成。
- 完成後給予代幣、解鎖、成長值或其他獎勵。

本專案不以「解鎖手機 App 使用時間」作為核心獎勵，而是改成：

- 讓小芽或樹苗成長。
- 讓家庭樹或個人樹累積成長度。
- 每個週期結算後，將虛擬成果轉入真實植樹池。
- 與企業 ESG、政府、環保團體或在地單位合作，轉化為真實種樹、社區綠化或原生植物復育。

## 使用者價值

### 長者

- 有自然、低壓力的活動理由。
- 任務貼近日常生活，不像被要求運動或被監督。
- 可以透過拍照、觀察、散步、記錄感受重新感受生活。

### 家庭成員

- 有自然互動入口，不只是問候或聊天。
- 可以遠端參與澆水、留言、鼓勵、協助任務。
- 家庭共同累積一棵樹的成長，形成共同目標。

### 無子女或獨居使用者

- 可使用個人樹、社區樹或公益森林。
- 不需要綁定家人，也能透過日常行動為環境累積貢獻。

### 合作單位

- 可以將使用者任務成果轉化為 ESG 參與紀錄。
- 可以支援真實植樹、社區綠化、原生植物復育或公益活動。

## 初步功能模組

- 生活任務：散步、拍花、看天空、喝水、呼吸、閱讀、專注、運動、去公園坐一下。
- 任務驗證：照片辨識、手動紀錄、GPS/步數資料、時間紀錄、家人確認。
- 樹木成長：任務完成後累積水滴、陽光、土壤或成長值。
- 家庭樹：家庭成員共同澆水、留言、陪伴與協作。
- 個人樹：沒有家庭群組的使用者也能獨立累積。
- 城市綠化地圖：顯示虛擬樹、公益植樹池、合作地點或城市綠化進度。
- 週期結算：例如每 30 天結算一次，將成熟樹轉換為真實植樹額度。
- ESG 後台：管理合作單位、植樹批次、任務成果、公益回饋證明。

## 地圖探索待討論

地圖探索可以保留，但需要重新定位。

可行方向：

1. 地圖不是遊戲主線，而是生活活動入口。
2. 地圖顯示附近可完成的低壓任務，例如公園、步道、社區花圃。
3. 地圖顯示城市裡的虛擬樹與真實植樹成果。
4. 地圖用於公益成果展示，而不是要求長者到處跑任務。
5. 對行動不便者提供居家任務，不強迫使用地圖。

## 暫定技術方向

- App：Flutter
- 後端：Firebase Auth、Firestore、Storage、Cloud Functions
- Web 後台：React/Vite 或 Firebase Hosting 靜態後台
- AI/辨識：初期可用照片標籤、簡單規則與人工確認，後期再接 AI 圖像辨識
