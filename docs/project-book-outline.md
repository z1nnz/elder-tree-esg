# 畢業專題計畫書實作版骨架

1. 緒論：高齡日常活動、家庭互動斷點與 ESG 可信度問題。
2. 文獻與競品：習慣任務、家庭科技、陪伴裝置、遊戲化與永續回饋。
3. 研究目標：低壓任務、跨世代共作、實體互動與可追溯成果。
4. 使用者研究：長者、家屬、社區與組織管理者四類角色。
5. 服務設計：任務、AI／人工覆核、樹木成長、家庭訊息與公益批次。
6. 系統架構：Flutter、NestJS、PostgreSQL、FastAPI、AWS IoT、ESP32。
7. 硬體設計：BOM、電路、外殼、人機互動、無相機／麥克風原則。
8. 資料庫與 API：關聯模型、冪等帳本、Device Shadow 與稽核紀錄。
9. 資訊安全與倫理：最小權限、位置／照片隱私、AI 限制、防漂綠。
10. 實作結果：App、Web、API、AI、IoT 與硬體端到端畫面。
11. 測試方法：單元、整合、HIL、golden set、SUS 與訪談。
12. 成本與商業模式：原型 BOM、雲端預算、組織專案服務。
13. 進度與分工：四條工作線、每週整合與風險緩衝。
14. 結論與未來工作：真實合作、長期養護、量產與多城市擴展。
