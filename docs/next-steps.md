## 大改版主線

這一輪不再先追更多特效，而是把已做好的能力收斂成一條每天真的能用的主產品：

登入 → 今日陪伴 → 選一件任務 → 完成 SELF_CHECK/TIMER/PHOTO_AI → 生命樹長出新葉 → 家人或陪伴者看見 → 關閉重啟後狀態仍存在。

### PR A — App V2 今日陪伴核心

- 新增 `GET /me/home`，用目前家庭、生命樹、任務、雷達、訊息、照片覆核與 capability 推導首頁摘要。
- App 首頁改成「今日陪伴」Hub：家庭、生命樹、下一個行動、任務卡堆疊、提醒與最近雷達任務集中呈現。
- 任務完成回饋統一為「生命樹長出新葉 +N」，並從後端刷新狀態，不做本地假加分。
- 任務卡統一狀態語言：可開始、進行中、待覆核、可重拍、已完成。
- 補 `docs/operations/app-v2-validation.md`，讓實機驗收可以照表操作。

### PR B — App 探索遊戲化

- 城市探索頁降低資訊密度：預設只顯示玩家位置、最近任務、任務半徑與開始探索 CTA。

- 加入輕量 2D 小葉靈／光點角色，跟隨任務靠近、解鎖、完成與休息狀態。
- 任務靠近、解鎖與完成做出清楚但不刺激的遊戲回饋；風格是溫柔冒險，不是競速排名。

### PR E — LINE Bot MVP

- LINE Bot 作為輔助入口，不取代 App。
- 第一版做每日提醒、任務推播、家人／陪伴者通知、照片待覆核通知與快速回覆。
- 綁定方式先用 App 產生一次性連結碼或 QR code。
- LINE Bot 不直接處理照片 Storage 上傳；PHOTO_AI 仍由 App 執行。
- 推播失敗不得影響 App 任務完成與成長值帳本。

## 驗收優先順序

1. iPhone / Android：登入後看到今日陪伴首頁。
2. SELF_CHECK：確認完成後，生命樹只增加一次。
3. TIMER：未達時間不可完成，達時間後可完成。
4. PHOTO_AI：植物或水杯 PASS，FAIL 可重拍，REVIEW 進入家人覆核。
5. 城市探索：開始探索後才使用定位，進入半徑才解鎖。
6. 關閉 App 重啟：家庭、任務、生命樹、訊息、照片覆核與探索狀態仍存在。
