# 實機展示驗收清單

這份清單用於比賽、展示或本地驗收前的快速檢查。

## 服務啟動

- API 已啟動並連到 Neon。
- AI verifier 已啟動，且 `GEMINI_API_KEY` 只存在本機環境。
- Firebase Storage rules 已部署。
- 後台可用平台管理員帳號登入。

## App 驗收

- 登入後能看到目前家庭、生命樹、一般任務與城市探索狀態。
- PHOTO_AI 任務按鈕顯示可拍照驗證；若環境缺少設定，會顯示可理解原因。
- 拍植物與水杯可完成任務。
- FAIL 不加成長值，可重新拍攝。
- REVIEW 會進入家人覆核，不會讓提交者覆核自己。
- 關閉 App 重啟後，任務狀態與生命樹成長值仍存在。

## 後台驗收

- 後台可看到照片 AI 狀態檢查。
- 任務覆核佇列能顯示待覆核照片。
- 雷達任務仍不能發布 `PHOTO_AI`。
- 一般使用者進入 admin API 會得到 403。

## 不納入本輪

- LINE Bot。
- 實體樹硬體擴充。
- 志工媒合。
- 雷達任務照片證據。
